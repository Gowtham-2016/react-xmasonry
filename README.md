# react-xmasonry

Simple, minimalistic and featured native masonry layout for React JS.

Example of Usage
----------------

```jsx
<XMasonry>
    <Card key="1">
        <h1>Card One</h1>
        <p>Any text</p>
    </Card>
    <Card key="2">
        <h1>Card Two</h1>
        <p>Any text</p>
    </Card>
</XMasonry>
```
