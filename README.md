# react-xmasonry

Simple and featured native masonry layout implementation for React JS.

Example
-------

```jsx
<XMasonry>
    <Card key=1>
        <h1>Card One</h1>
        <p>Any text</p>
    </Card>
    <Card key=2>
        <h1>Card Two</h1>
        <p>Any text</p>
    </Card>
</XMasonry>
```
